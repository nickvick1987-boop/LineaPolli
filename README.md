# LineaPolli

LineaPolli — Production Tracker

**Autore:** Vick

## Contenuti
- `LineaPolli.html` — versione standalone (aprire in browser)
- `LineaPolli-Vite/` — progetto Vite + React + Tailwind (sorgenti)

## Requisiti (per la versione Vite)
- Node.js 18+ raccomandato

## Setup (Vite)
```bash
cd LineaPolli-Vite
npm install
npm run dev
```

## Salvataggio
I dati vengono salvati automaticamente nel `localStorage` del browser **quando cambiano**: impostazioni, stato, log pause e conteggio pezzi.
